## Selected work

| Project | The interesting part | Built with |
|---|---|---|
| [**image-segmentation**](https://github.com/doorukb/image-segmentation) | Image segmentation from scratch via max-flow/min-cut over a pixel graph — no CV libraries doing the thinking | ![Python](https://img.shields.io/badge/-Python-3572A5?style=flat-square&logo=python&logoColor=white) |
| [**multicore-task-scheduler**](https://github.com/doorukb/multicore-task-scheduler) | A scheduler engine simulating a multi-core CPU, built on real OS scheduling concepts | ![C](https://img.shields.io/badge/-C-555555?style=flat-square&logo=c&logoColor=white) |
| [**peg-interpreter**](https://github.com/doorukb/peg-interpreter) | Tree-walking interpreter for a JS-like language — closures, recursion, and block scoping that actually behave | ![TypeScript](https://img.shields.io/badge/-TypeScript-3178c6?style=flat-square&logo=typescript&logoColor=white) |
| [**markov-music-engine**](https://github.com/doorukb/markov-music-engine) | Hierarchical Markov chains that learn structure from music and generate their own | ![Python](https://img.shields.io/badge/-Python-3572A5?style=flat-square&logo=python&logoColor=white) |
| [**baby-garbage-collector**](https://github.com/doorukb/baby-garbage-collector) | A dependency-free mark-and-sweep garbage collector for C | ![C](https://img.shields.io/badge/-C-555555?style=flat-square&logo=c&logoColor=white) |
| [**smart-syllabus-scanner**](https://github.com/doorukb/smart-syllabus-scanner) | Document-intelligence pipeline: raw PDF/PNG/TXT syllabus in, structured schedule out | ![Python](https://img.shields.io/badge/-Python-3572A5?style=flat-square&logo=python&logoColor=white) |
